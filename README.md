## 简介
>本项目对数据主键符合帕累托分布(二八原则)的Cache缓存进行优化。
为优化对热点数据的缓存，本项目设计了两个LRU队列，一个LRU存储了数据，另一个统计数据键的热度变化。

![avatar](https://github.com/boostlearn/go-safe-cache/raw/master/doc/safe_cache.png)
